# Random Quote Generator 🎲

**Автор:** Мускалов Артур

## Описание проекта

Random Quote Generator — это графическое приложение на Python с интерфейсом Tkinter, которое позволяет:
- Генерировать случайные цитаты из предопределённого списка
- Добавлять новые цитаты с валидацией ввода
- Просматривать историю сгенерированных цитат
- Фильтровать историю по автору и теме
- Сохранять все данные в JSON файл
- Загружать данные при запуске приложения

## Функциональность

### Основные возможности
1. **Генерация случайных цитат** — при нажатии на кнопку выбирается случайная цитата из базы
2. **Добавление новых цитат** — форма для добавления цитат с проверкой на пустые поля
3. **История цитат** — все сгенерированные цитаты сохраняются с отметкой времени
4. **Фильтрация** — возможность фильтровать историю по автору и теме
5. **Сохранение данных** — автоматическое сохранение в JSON файл

### Валидация ввода
- Проверка на пустые строки при добавлении новых цитат
- Проверка на существование файла данных при загрузке
